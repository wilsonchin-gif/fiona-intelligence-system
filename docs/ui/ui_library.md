## 1. UI 定位

Fiona 视觉风格参考：

- Bloomberg Terminal
- Kaiko
- Glassnode
- AI Analyst Dashboard

避免：

- 财经公众号风格。
- 新闻海报风格。
- 单图信息过载。

## 2. 色彩

- US Market：蓝色。
- China Market：红色。
- Crypto Market：黄色 + BTC Logo。
- RWA Market：绿色。
- Fiona's View：底部醒目区域。

## 3. 字体

- 标题：36px+
- 正文：22px+
- 核心数字：28px+

## 4. 图片规格
