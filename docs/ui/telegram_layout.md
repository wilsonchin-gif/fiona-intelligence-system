## 1. 文字版结构

所有 Telegram 内容以中文为主，保留必要英文标题。

必须包含：

- 标题。
- 更新时间。
- 核心内容模块。
- Fiona's View。
- Disclaimer。

## 2. 内容规则

禁止：

- 买入。
- 卖出。
- 加仓。
- 减仓。
- 目标价。
- 必涨。
- 必跌。

允许：

- 观察。
- 确认。
- 风险。
- 情景。
- 资金流。
- 叙事。
- 市场状态。
