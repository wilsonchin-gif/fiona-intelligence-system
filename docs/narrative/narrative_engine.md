# Fiona Narrative Engine

版本：V1.0.0  
状态：Active  
负责人：Wilson  
更新时间：2026-06-26

## 1. 定位

Narrative Engine 负责识别市场正在讲什么，而不是简单记录发生过什么。

## 2. 输出类型

- Current Narrative
- Emerging Narrative
- Fading Narrative
- False Narrative

## 3. 使用场景

- Daily 自动引用 Current / Emerging Narrative。
- Weekly 输出 Narrative Ranking。
- False Narrative Watchlist 用于识别高热度、弱资金、低持续性的主题。
