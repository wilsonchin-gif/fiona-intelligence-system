# Fiona Narrative Scoring

版本：V1.0.0  
状态：Active  
负责人：Wilson  
更新时间：2026-06-26

## 1. Narrative Score

评分维度：

- 频率。
- 资金确认。
- 跨市场影响。
- 持续性。
- 情报价值。

## 2. 分级

- 80+：Current Narrative
- 60-79：Emerging Narrative
- 40-59：Watchlist Narrative
- <40：Fading Narrative

## 3. False Narrative

特征：

- 热度高。
- 资金弱。
- 持续性差。
- 无法形成主线。
