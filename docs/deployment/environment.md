# Fiona Environment Variables

版本：V1.0.0  
状态：Active  
负责人：Wilson  
更新时间：2026-06-26

## 必需变量

```env
TELEGRAM_BOT_TOKEN=
TELEGRAM_GROUP_ID=
WILSON_SEND=1
WILSON_TIMEZONE=Asia/Manila
WILSON_INTERVAL_MINUTES=240
FIONA_ALERT_ENABLED=0
FIONA_ALERT_DRY_RUN=1
```

## 可选兼容变量

```env
TELEGRAM_CHAT_ID=
TELEGRAM_CHANNEL_ID=
FIONA_SEND=
FIONA_SEND_TELEGRAM=
FIONA_TIMEZONE=
FIONA_RUNTIME_INTERVAL_MINUTES=
```

## 不推荐作为生产默认

- `FIONA_SEND`
- `FIONA_SEND_TELEGRAM`
- `TELEGRAM_CHANNEL_ID`
